Projects

TaskFlow – Simple To-Do Web App

A minimalist task manager built with HTML, CSS, and JavaScript TaskFlow allows users to create, delete, and mark tasks as complete with a clean, responsive interface. The app uses localStorage to save tasks between sessions, showing frontend skills and user-focused design.

TaskFlow - Simple To-Do WebApp

GitGuide – Interactive Git Tutorial

A command-line based Git tutorial built for beginners GitGuide walks users through basic Git commands like init, status, add, commit, and branching using Node.js. It's a learning tool to help others understand version control while demonstrating your own comfort with CLI tools and scripting.

GitGuide – Interactive Git Tutorial

FactoCalc – Recursive Factorial Calculator

A simple number tool using JavaScript recursion and input validation This project takes a number input from the user and calculates its factorial recursively. It includes helpful error messages and uses clean, reusable functions. Great for highlighting algorithmic thinking and function design.
